Raspberry Pi Memorándum

LA META

Esta página es un memorándum de las configuraciones de Raspberry Pi para establecer un servidor de hogar en el entrono de red por axtel x-tremo. El objetivo es realizar los siguientes:

  • Operar Raspberry Pi por Windows afuera de LAN a través de SSH
  • Página de web por WordPress en Raspberry Pi
  • Otros servicios útiles como FTP, NAS y DLNA
